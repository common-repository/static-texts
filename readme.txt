== Description ==
Static Texts plugin helps you put short texts on your WordPress site. This can be useful for some kinds of texts such as highlights, feature reviews, welcome/introduction texts, contact information and more. Static Texts can be useful for short messages, video/image embed or any other HTML contents that are not suitable for a post or page.

**Features**

1. Support texts, images, video or other html contents
2. Built-in function to integrate into your WordPress theme
3. Use Custom Post Type to organize your static items
4. Simple, light-weighted plugin
5. Support qTranslate plugin for multiple languages


== Installation ==
1. Upload Static Text to the /wp-content/plugins/ directory.
2. Activate the plugin through the Plugins menu item in the WordPress Dashboard.
3. Once the plugin is activated, add your static items inside WordPress Admin and use built-in function to display.
